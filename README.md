# Дискуссионная платформа (Форум)

Веб-приложение для создания обсуждений и обмена сообщениями. Разработано на MERN-стеке (MongoDB, Express, React, Node.js).

## 🚀 Функционал

- Создание тем обсуждений
- Просмотр списка всех тем
- Добавление сообщений в темы
- Удаление тем и сообщений
- Автоматическое сохранение данных в MongoDB
- Адаптивный дизайн

## 🛠 Технологии

Backend:
- Node.js
- Express.js
- MongoDB + Mongoose
- CORS, dotenv

Frontend:
- React 18
- React Router DOM
- Axios
- Vite
